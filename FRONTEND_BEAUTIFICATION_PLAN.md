# 前端美化清单 — 企业办公风格

> 参照 D:\demo_test 设计风格，为 D:\all_ds 前端进行企业级美化。
> **原则：只美化 UI，不改动任何后端逻辑和功能交互。**

---

## 一、设计风格定义

### 1.1 风格关键词

| 关键词 | 说明 |
|--------|------|
| 企业办公 | 专业、冷静、数据优先的商业技术风 |
| 深色科技 | 深色背景 + 蓝色品牌色 + 粒子动效 |
| 简约克制 | 微动效、无花哨装饰、信息密度优先 |

### 1.2 设计 Token（CSS 变量）

从 demo_test 的 tokens.css 提取并适配暗色主题：

| Token | 值 | 用途 |
|-------|-----|------|
| `--color-brand` | #2A6BFF | 品牌主色（蓝） |
| `--color-brand-light` | #5B8DFF | 品牌亮色 |
| `--color-brand-dark` | #1A54E0 | 品牌暗色 |
| `--bg-main` | #0c0c0d | 主背景（当前已用） |
| `--bg-card` | #1a1a1e | 卡片背景 |
| `--bg-sidebar` | #121215 | 侧边栏背景 |
| `--text-primary` | #f5f5f5 | 主文字 |
| `--text-secondary` | #a1a1aa | 次要文字 |
| `--border-color` | #27272a | 边框颜色 |
